# Users App

This folder contains the `users-app` Express starter. It includes:

- an Express server entrypoint
- MongoDB / Mongoose connection setup
- users and transactions API routes
- Babel, ESLint and Prettier configuration
- Installed packages for testing (`jest`, `supertest`)

## Getting started

1. From the folder, run `npm install`.
2. Make sure MongoDB is running locally
3. Start the dev server with `npm run dev`.
4. Run tests with `npm test`.
